RS-232 Ports

The RS-232 (SIO) ports can be written to and read from using the diagnostics. All transfers are at 8 data bits, no parity and 1 stop bit. This cannot be changed in the diagnostics.

This is accomplished using the following QuickUSB Library functions:
QuickUsbSetRS232Baud - Sets the baud rate of both ports.
QuickUsbWriteRS232 - Writes a block of binary data to an RS-232 port.
QuickUsbReadRS232 - Reads a block of binary data from an RS-232 port. Both ports have interrupt-driven buffers inside the FX2 and this routine reads those buffers.
QuickUsbFlushRS232 - Flushes a port's interrupt driven receive buffer.
QuickUsbGetNumRS232 - Returns the number of characters waiting in a receive buffer.

The RS-232 tab is shown below:

Port
Selects either SIO-0 or SIO-1.

Baud Rate
Selects the baud rate for both the RS-232 ports.

Terminal Window
You can type text into the terminal window and it will be sent to the selected SIO port at the selected baud rate. Likewise, data received on the SIO port will be displayed in the terminal window surrounded with parenthesis.
