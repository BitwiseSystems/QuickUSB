I/O Ports

The QuickUSB Diagnostics for the Cypress EZ USB-FX2 gives you the power to directly control 3 8-bit parallel ports from the comfort of your PC without writing any firmware.

This is accomplished using the following QuickUSB Library functions:
QuickUsbWritePortDir - Specifies the direction (input or output) of each pin of a port
QuickUsbWritePort - Specifies the value of an output pin.
QuickUsbReadPort - Reads the current value of all the pins of a port. Outputs return the output value and inputs return the value on the pin.

The I/O port tab is shown below:

Port A - P2, Pins 12-19
Bit 7 corresponds to pin 12 and bit 0 corresponds to pin 19.

Pin Direction
Each checkbox corresponds to the I/O direction of a pin on the port. A check means that the pin in an output and an empty checkbox means that the pin is an input.

Pin Value
Each checkbox corresponds to the output value of a pin on the port. A check means that the output is high and an empty checkbox means that the pin is low.

Read
Reads both the direction and value of the pins of a port.
